Info

  1. The product_ID must be unique
  2. The icon with the little yellow pen means that you can update some of the product properties ( only name, price, amount )
  3. The icon with the little red bin means that you can delete that product
  4. After you have changed a property of a product you must press the 'Enter'-key on your keyboard so that it will update
  5. You can't update the product_ID, you must delete the entire product in case that you change the product_ID of a specific product and then re-enter the name, price & amount
